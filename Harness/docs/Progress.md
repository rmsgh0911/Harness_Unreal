# Progress

이 문서는 사람이 빠르게 현재 진행 상황을 확인하기 위한 한국어 대시보드입니다.

Harness의 다른 운영 문서는 이식 안정성을 위해 기본적으로 영어로 작성합니다. 이 파일만 사람 확인용으로 한국어를 사용합니다.

상세 작업 로그는 `Harness/cycles/`, 최신 기술 상태는 `Harness/state.md`, 남은 일과 결정 사항은 `Harness/next.md`를 기준으로 봅니다.

## 현재 목표

- 작성 필요: 지금 프로젝트에서 가장 중요한 목표를 1-3개만 적습니다.

## 현재 상태

- 작성 필요: 확정된 현재 상태만 짧게 적습니다.

## 최근 완료

- 작성 필요: 사람이 알아야 하는 완료 항목만 짧게 적습니다.

## 사람 확인 필요

- 작성 필요: PIE, 입력 감각, 애니메이션, 에셋 상태처럼 사람 확인이 필요한 항목을 적습니다.

## 다음 결정

- 작성 필요: 방향 전환, 범위 조정, 수동 판단이 필요한 항목을 적습니다.
