# Progress

이 문서는 사람이 현재 진행 상황을 빠르게 확인하기 위한 현황판이다.

Harness의 다른 운영 문서는 이식 안정성을 위해 영어로 작성한다. 이 파일만 기본적으로 한국어로 작성한다.

상세 작업 로그는 `Harness/cycles/`, 에이전트 인수인계용 최신 기술 상태는 `Harness/state.md`, 실제 남은 일과 리스크는 `Harness/next.md`를 기준으로 한다.

## 현재 목표

- 작성 필요: 지금 프로젝트에서 가장 중요한 목표를 1-3개만 적는다.

## 현재 상태

- 작성 필요: 확정된 현재 상태만 짧게 적는다.

## 최근 완료

- 작성 필요: 사람이 알아야 할 완료 항목만 짧게 적는다.

## 사람이 확인할 것

- 작성 필요: PIE, 입력 감각, 애니메이션, 에셋 저장 여부처럼 사람 확인이 필요한 항목을 적는다.

## 다음 결정
